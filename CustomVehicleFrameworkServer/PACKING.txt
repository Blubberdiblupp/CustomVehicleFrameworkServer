CUSTOM VEHICLE FRAMEWORK SERVER BOOTSTRAP
=========================================

Pack this source directory as:

  @CustomVehicleFrameworkServer\addons\CustomVehicleFrameworkServer_Bootstrap.pbo

Required Addon Builder settings:

  Binarize: OFF
  Addon prefix: profiles\CustomVehicleFramework_Generated

The addon prefix must match exactly. The PBO must keep config.cpp as text;
otherwise DayZ cannot replace it with the generated profile config through
-filePatching.

This PBO is server-only. Load it with:

  -serverMod=@CustomVehicleFrameworkServer

Also launch the server with -filePatching. Do not put the generated profile
directory itself in -serverMod.

The normal @CustomVehicleFramework mod remains in -mod on server and clients.
